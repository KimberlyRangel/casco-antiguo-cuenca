# Casco Antiguo de Cuenca - Web Premium

Este proyecto es una página web moderna, elegante y bilingüe dedicada al Casco Antiguo de Cuenca, Patrimonio de la Humanidad por la UNESCO.

## ✨ Características

- **Diseño Premium**: Estética de lujo con tonos negros y dorados, tipografía cuidada y efectos de glassmorphism.
- **Bilingüe (ES | EN)**: Sistema de traducción nativo (Español e Inglés) con persistencia de idioma.
- **Experiencia Interactiva**: Animaciones al hacer scroll y carga progresiva de contenido.
- **Compartir**: Botón de compartir integrado con la Web Share API y soporte para copiar al portapapeles.
- **Navegación**: Enlaces directos a Google Maps y secciones detalladas de historia y gastronomía.

## 🛠️ Tecnologías

- **HTML5**: Estructura semántica.
- **CSS3**: Diseño responsivo y animaciones personalizadas.
- **JavaScript (Vanilla)**: Lógica de traducción, compartir y animaciones.

## 🖼️ Créditos

Las imágenes utilizadas muestran la belleza de la ciudad de Cuenca:
- Casas Colgadas
- Puente de San Pablo
- Catedral de Santa María
- Gastronomía típica (Morteruelo, Zarajos, Ajoarriero)
